# 🗺️ AR Indoor & Outdoor Navigation System

## 📌 Overview
This project is a **real-time AR navigation system** that overlays live directional paths onto the physical world using Unity, Mapbox, and SLAM.  
It is designed to improve navigation both indoors and outdoors, especially in GPS-weak areas.

---

## 🎯 Features
- 🧭 Real-time AR path rendering in Unity  
- 📍 Indoor & outdoor support  
- 📡 Improved positioning accuracy with sensor fusion (IMU + SLAM)  
- 👥 Multi-user route sharing using persistent cloud anchors  
- 📱 Built for ARCore/ARKit devices  

---

## 🛠️ Tech Stack
- **Game Engine:** Unity 3D  
- **Languages:** C#  
- **AR:** AR Foundation, SLAM, Google Cloud Anchors  
- **Maps:** Mapbox SDK  
- **Algorithms:** Sensor fusion (IMU + SLAM)
